## Version

Developpement en python 3.12.1

## Clone
    
```bash
git clone git@github.com:AlexTraveylan/train_http.git
cd train_http
```

## Installation

```bash
python -m env venv
env\Scripts\activate
pip install -r requirements.txt
```
